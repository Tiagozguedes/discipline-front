# Discipline - Frontend

Plataforma de monitoramento de disciplina pessoal. Frontend React + TypeScript + Vite com tema escuro XP Investimentos.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Recharts (graficos)
- Lucide React (icones)

## Funcionalidades (Phase 1 MVP)

- Dashboard com Discipline Score e widgets
- Relogio Pomodoro com sessoes de foco/pausa
- Gerenciador de Tarefas (Kanban: A Fazer / Em Progresso / Concluido)
- Calendario/Agenda com visualizacao mensal
- Rastreador de Habitos com heatmap semanal e streaks
- Modulo Financeiro: transacoes, investimentos, relatorios visuais

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
